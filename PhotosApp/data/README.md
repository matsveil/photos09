# Stock Photos Directory

This directory should contain 5-10 stock photos for the application.

The stock photos should be named:
- stock1.jpg
- stock2.jpg
- stock3.jpg
- stock4.jpg
- stock5.jpg

These photos will be automatically loaded into the "stock" user's "stock" album on first run.

**Important:** Please add actual image files (JPG, PNG, GIF, or BMP format) to this directory before running the application for the first time.

For testing purposes, you can use any low/medium resolution images (to keep the repository size small).
